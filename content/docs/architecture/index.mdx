## BlockMind System Landscape

BlockMind consists of three primary systems:

- **BlockMind Platform** — TanStack Start web app on Vercel. Portfolio tracking, chat UI, real-time data sync.
- **Elite Agent System** — One Fly Machine per user running an OpenClaw gateway. Per-user AI with persistent memory, tools, and browser access.
- **Agent Infrastructure** — Inngest-driven provisioning, template distribution, machine pooling, health monitoring.

### External Dependencies

| System | Purpose |
|--------|---------|
| PlanetScale | Postgres database (Drizzle ORM) |
| Fly.io | Agent compute — one dedicated machine per user |
| Vercel | Platform hosting + serverless functions |
| Inngest | Durable workflow orchestration |
| Electric SQL | Durable Streams — real-time SSE event transport |
| Browserbase | Cloud browser sessions for agents |
| Tigris S3 | Artifact storage (screenshots, files) |
| Moralis | Blockchain/DeFi data provider |
| AgentMail | Per-agent email addresses |
