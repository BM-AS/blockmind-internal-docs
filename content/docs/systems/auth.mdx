## Authentication System

BlockMind uses **Better Auth** for authentication and session management.

### Key Features

- Email/password authentication
- OAuth providers (Google, GitHub)
- Session management with secure cookies
- Role-based access control (user, admin)

### Architecture

Auth is handled by the `auth` container in the platform, sitting between the API layer and the database. All authenticated routes go through Better Auth middleware.
