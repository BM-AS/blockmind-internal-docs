# Review Context — BlockMind Internal Docs Site

## What This Is
A Fumadocs v16 + Next.js 16 internal documentation site for BlockMind's architecture. It documents:
- C4 architecture (Level 1-3) via Structurizr DSL → D2 → SVG pipeline
- 7 architecture pages: System Landscape, Platform, Elite, Infrastructure, Chat Flow, Provisioning Flow, Credit System
- 1 ADR, 1 Systems placeholder, 1 Agents placeholder
- Custom D2Diagram component for embedding SVGs
- Build script with CI/Vercel detection (uses pre-committed SVGs)

## Target Audience
- **Product owners** (CEO Oskar, non-technical) — should understand system landscape + high-level flows
- **Developers** (Philip CTO, Trym full-stack) — should get enough context to understand architecture decisions
- **AI agents** (Odin, Noldus, Argos) — agents write content, humans read/comment
